# Contributing to Weather Theme Extension

Thank you for your interest in contributing! Here's how to get started:

## Getting Started

1. **Fork** the repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/weather-theme-extension.git
   ```
3. Create a new branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```

## Making Changes

- Keep your changes focused and scoped to one feature or bug fix per PR
- Follow the existing code style
- Add clear comments to non-obvious logic
- Update the `README.md` if your changes affect setup or usage

## Submitting a Pull Request

1. Commit your changes with a clear message:
   ```bash
   git commit -m "Add: brief description of what you did"
   ```
2. Push to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```
3. Open a Pull Request against the `main` branch
4. Describe what you changed and why in the PR description

## Areas to Contribute

Check the **To-Do** section in the README for open tasks:
- Background gradients per weather condition
- City selection option
- More granular weather condition themes

## Reporting Bugs

Open a GitHub Issue with:
- Steps to reproduce the bug
- Expected vs actual behavior
- Browser version and OS
